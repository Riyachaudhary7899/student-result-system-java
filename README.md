# Student Result Management System

## What is this?
This is a simple Java console-based project.
It takes student details and marks, calculates total and percentage, and shows PASS or FAIL.

---

## Features
- Takes student name and roll number
- Accepts marks of three subjects
- Calculates total and percentage
- Displays PASS or FAIL result

---

## How to Save 
1. Create a file named:
   StudentResultSystem.java
2. Paste the Java code into this file
3. Save the file

---

## How to Run 
1. Open terminal / command prompt
2. Compile the program:
   ```bash
   javac StudentResultSystem.java
3. Run the program:
   ```bash
   java StudentResultSystem
